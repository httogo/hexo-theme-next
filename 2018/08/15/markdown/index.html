HEXO-MarkDown使用指南

Posted on 2018-08-15 Edited on 2019-11-20
Hexo博客写作使用指南.

        


        
        


建立标题
在MakeDown中标题的写法很简单，只需在顶格写一个”#”号，然后空一格加上标题内容就可以了：


# 一级标题
次级标题的写法
二级标题的写法类似于一级标题，只是’#’号变成了两个：


## 二级标题
与此类似，三级、四级…标题只需加上对应数目的’#’号就可以了


列表
无序列表
无序号列表写法为顶格”-“后空一格并加上列表元素内容：


- 语文
- 数学
- 英语

有序列表
有序列表序号加”.”空一格然后添加内容即可：


1. 橘子
2. 橙子
3. what
效果如下：

    2. 

  2. 橙子
    3. 

  3. what
    4. 



超链接
在中括号内填写要显示出来文字内容，后面跟一个小括号填写链接地址。


写法如下：


[超链接](https://httogo.github.io/)

这是一个超链接：超链接。
